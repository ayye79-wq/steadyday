# SteadyDay

**SteadyDay** is a calm, private companion for daily self-care.

It helps you:
- Track morning and evening medications  
- Set gentle time targets and reminders  
- Log blood sugar readings  
- Keep short daily notes  
- See your history over time  

All data stays on your device.  
No accounts. No cloud. No tracking.

> Helping you care for yourself, one day at a time.

Live app:  
https://ayye79-wq.github.io/steadyday/

---

Built with simple HTML, CSS, and JavaScript.  
Designed to be lightweight, fast, and stress-free.
